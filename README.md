# 🧠 ADHD Disease Detector

An AI-powered system that predicts the likelihood of Attention Deficit Hyperactivity Disorder (ADHD) using machine learning techniques. This project aims to assist in early detection and provide data-driven insights for better understanding behavioral patterns.

---

## 🚀 Project Overview

ADHD is a neurodevelopmental disorder that affects attention, focus, and impulse control. Early detection plays a crucial role in proper diagnosis and treatment.

This project leverages **Machine Learning** to analyze input data and predict whether a person may have ADHD tendencies.

---

- Detect ADHD using machine learning models
- Provide early insights for better intervention
